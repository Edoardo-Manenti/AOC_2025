# AOC_2025
Repository containing my solutions to the Advent Of Code event for the year 2025

## REPOSITORY Structure

```
AOC_2025/
├── .gitignore
├── day_01/
│   ├── aoc.py # Python solution for the daily problems
│   ├── input.txt # Personal input for the daily problem (NOTE: this file will not be pushed to the public repo)
│   ├── run.sh # Bash script to run/test the python solution
│   ├── NOTES.md # (Optional: Notes describing the approach to the solution)
│   └── README.md # Problem of the day description
├── Template/
│   ├── aoc.py # Python template solution with support for testing on examples and basic benchmarking of the solution
│   └── run.sh # Bash script to run/test the python solution of the day
├── dl_potd.sh # Bash script that downloads the problem of the day and sets up the relative directory leveraging the aoc_utils project (https://github.com/Edoardo-Manenti/aoc-utils)
└── README.md # The file you are reading right now
```
